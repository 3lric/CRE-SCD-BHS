=====================================================
           CRE SCD Editor Suite v0.6b by 3lric
=====================================================

Thank you for downloading the CRE SCD Editor Suite! This program is a powerful tool designed for modding and editing SCD files from the classic Resident Evil games. 

=====================================================
                      Features
=====================================================

- **Opcode Editing**: Modify opcode information for Resident Evil 1, Resident Evil 1.5, Resident Evil 2, and Resident Evil 3.
- **Theme Customization**: Personalize the look of the program with built-in or custom themes.
- **Backup & Restore**: Automatically creates backups to keep your original files safe.
- **Dynamic Reloading**: Instant changes to themes and opcode data without restarting the program.
- **Undo/Redo**: Track and revert or reapply changes as needed.
- **Drag-and-Drop Support**: Easily rearrange opcodes for better organization.

=====================================================
                  Installation Instructions
=====================================================

To install and run the program, follow these steps:

1. **Executable**: 
   - Download and extract the latest version of the program.
   - Run `main.exe`.

2. **From Source**:
   - Make sure you have Python 3.12 or higher installed.
   - Install required dependencies using `pip install -r requirements.txt`.
   - Run `main.py` using Python.

=====================================================
              Supported Game Versions
=====================================================

- Resident Evil 1 (RE1)
- Resident Evil 1.5 (RE1.5)
- Resident Evil 2 (RE2)
- Resident Evil 3 (RE3)
